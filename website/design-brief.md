# Vutler — Design Brief

## Brand Identity

**Positioning:** The AI-native workspace. Office 365 for agents.
**Personality:** Bold, technical, trustworthy. Not corporate-boring, not startup-chaotic.
**Feeling:** "This is where serious AI work happens."

## Colors

### Primary Palette
- **Deep Navy:** `#0A1628` — backgrounds, authority
- **Electric Blue:** `#2563EB` — CTAs, links, accent
- **White:** `#FFFFFF` — text on dark, cards

### Secondary
- **Swiss Red:** `#DC2626` — sparingly, Swiss identity nod
- **Emerald:** `#10B981` — success states, "active" indicators
- **Slate:** `#64748B` — secondary text
- **Ghost White:** `#F8FAFC` — section backgrounds

### Dark Mode (Default)
The site should default to dark mode. AI/dev tools are dark. Respect the culture.

## Typography

- **Headlines:** Inter or Satoshi — bold, geometric, modern
- **Body:** Inter — clean, highly readable
- **Code/Mono:** JetBrains Mono — for feature labels, tech specs
- **Sizes:** Large headlines (48-64px hero), generous line-height (1.6 body)

## Tone of Voice

- Direct. No fluff.
- Technical credibility without jargon overload
- Confident but not arrogant
- Occasional wit (headlines, CTAs)
- Always conversion-oriented

**Do:** "Your agents collaborate. You approve what matters."
**Don't:** "Leverage our cutting-edge AI-powered synergistic platform."

## Layout Principles

- **Generous whitespace** — let features breathe
- **Card-based feature sections** — icon + title + 2-line description
- **Full-width hero** with animated/illustrated workspace mockup
- **Sticky nav** with CTA always visible
- **Mobile-first** responsive design

## Visual Elements

- **Icons:** Lucide or Phosphor — consistent line style
- **Illustrations:** Minimal, geometric. Agent "characters" as abstract shapes, not cartoon robots.
- **Screenshots:** Actual app screenshots from app.vutler.ai (with polish)
- **Gradients:** Subtle dark gradients (navy → deeper navy). No rainbow chaos.
- **Borders:** Subtle `1px` borders with low-opacity whites on dark backgrounds

## Inspiration Sites

- **linear.app** — clean, dark, developer-focused
- **vercel.com** — bold typography, minimal layout
- **raycast.com** — feature showcasing done right
- **notion.so** — approachable productivity feel
- **supabase.com** — open source + product polish balance

## Animations

- Subtle fade-in on scroll
- Smooth hover states on cards and buttons
- No heavy animations — performance matters
- Consider: typing animation for agent chat demo on hero

## Logo Direction

- Wordmark: "Vutler" in geometric sans-serif
- Icon: Abstract "V" that suggests connection/collaboration
- Works on dark and light backgrounds
- Swiss cross subtle integration possible (but not mandatory)
