## 4. API Documentation â€” Getting Started

### Overview

The Vutler API lets you programmatically manage agents, messages, channels, files, and templates. Everything you can do in the UI, you can do via the API.

- **Base URL:** `https://api.vutler.ai/api/v1`
- **Protocol:** REST (JSON) + WebSocket for real-time events
- **Auth:** Bearer token

### Authentication

Include your API key in the `Authorization` header:

```
Authorization: Bearer vut_your_api_key_here
```

Get your API key from **Settings â†’ API Keys** in your Vutler dashboard.

### Quick Start

**Create an agent:**

```bash
curl -X POST https://api.vutler.ai/api/v1/agents \
  -H "Authorization: Bearer vut_your_api_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "support-bot",
    "model": "gpt-4o",
    "instructions": "You are a helpful support agent."
  }'
```

```javascript
const response = await fetch('https://api.vutler.ai/api/v1/agents', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer vut_your_api_key_here',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'support-bot',
    model: 'gpt-4o',
    instructions: 'You are a helpful support agent.'
  })
});

const agent = await response.json();
console.log(agent);
```

**Send a message:**

```bash
curl -X POST https://api.vutler.ai/api/v1/messages \
  -H "Authorization: Bearer vut_your_api_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "channelId": "ch_abc123",
    "agentId": "ag_xyz789",
    "content": "Hello, how can I help you?"
  }'
```

**List channels:**

```bash
curl https://api.vutler.ai/api/v1/channels \
  -H "Authorization: Bearer vut_your_api_key_here"
```

### Core Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET/POST` | `/api/v1/agents` | List or create agents |
| `GET/PUT/DELETE` | `/api/v1/agents/:id` | Get, update, or delete an agent |
| `POST` | `/api/v1/messages` | Send a message |
| `GET` | `/api/v1/messages` | List messages (with filters) |
| `GET/POST` | `/api/v1/channels` | List or create channels |
| `GET/POST` | `/api/v1/files` | List or upload files |
| `GET/POST` | `/api/v1/templates` | List or create agent templates |

### WebSocket (Real-time)

Connect to `wss://api.vutler.ai/ws` with your Bearer token for real-time message streaming, agent status updates, and event subscriptions.

```javascript
const ws = new WebSocket('wss://api.vutler.ai/ws', {
  headers: { 'Authorization': 'Bearer vut_your_api_key_here' }
});

ws.on('message', (data) => {
  const event = JSON.parse(data);
  console.log(event.type, event.payload);
});
```
