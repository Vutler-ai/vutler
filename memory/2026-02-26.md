- The index.js uses `module.exports = { app, server }` not `module.exports = app`
- Sprint 7 routes are AFTER module.exports (unusual but works)
- Best approach: append new routes at end of file
- `routes/chat.js` expects `message` (string), `agents.js` expects `messages` (array) — different formats
