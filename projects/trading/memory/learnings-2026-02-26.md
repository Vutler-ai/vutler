# Trading Learnings — February 26, 2026

## Learning #1: Consolidation After Big Moves
**Context:** QQQ +23.8% yesterday, then -0.5% today  
**Insight:** After a big up day (+20%+), a flat or small down day (-0.5% to -1%) is normal consolidation and healthy. This is not underperformance - it's the market digesting gains.  
**Rule:** Don't confuse "not crashing" with "not performing". Consolidation = strength.  
**Tags:** psychology, consolidation, momentum

## Learning #2: Position P&L vs Daily P&L
**Context:** Portfolio showing +€1,746 total but -€13 today  
**Insight:** Must distinguish between:
- **Position P&L** (entry → now) = +€1,746 (+17.5% since start)
- **Daily P&L** (yesterday close → today close) = -€13 (-0.1% today)

The big gain was yesterday, not today. Understanding this prevents false signals and emotional reactions.  
**Rule:** Track both metrics separately. Position P&L = strategic view, Daily P&L = tactical view.  
**Tags:** metrics, portfolio-management, P&L-tracking

## Learning #3: When Forecasts Are Exceeded
**Context:** QQQ delivered +23.8% vs +3-5% forecast (4-8x faster)  
**Insight:** When a position exceeds forecast by 4-8x, set mental profit-taking alerts but let winners run.  
**Rule:** Hold unless position hits +30% total, then consider trimming 50% of position. Don't kill momentum too early, but preserve gains at parabolic levels.  
**Framework:**
- 0-10% above forecast → let it run fully
- 10-20% above forecast → watch closely, no action yet
- 20-30% above forecast → set trailing stop or mental alert
- 30%+ total gain → trim 50%, lock in profits, let rest ride

**Tags:** profit-taking, position-sizing, risk-management

## Learning #4: Entry-Day Volatility Is Noise
**Context:** GLD down -0.5% on entry day  
**Insight:** Entry-day volatility of ±1% is normal noise. Don't react to small moves on day 1.  
**Rule:** Long-term thesis (e.g., JPMorgan $6,300 target = +23% by year-end) requires patience. Only act if stop-loss (-5%) is breached. Entry day = watch, don't trade.  
**Tags:** patience, long-term-investing, gold, noise-vs-signal
